How to build MediaConch
-----------------------
Download and unzip the src01*.zip. Enter the new directory created.
Download and unzip the buildenv01*.zip.

Move the content of the buildenv01 directory to the current directory.

Execute the .bat file corresponding to your Visual Studio version.
In Visual Studio interface, select Release as target then build the solution.
